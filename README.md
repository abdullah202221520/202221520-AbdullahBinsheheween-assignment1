# Portfolio Website: SWE363 Assignment 1

## Project Description

A responsive personal portfolio website built with HTML, CSS, and JavaScript. It includes a hero section, an about section, a projects showcase, a contact form, and a dark mode toggle. The site is structured for accessibility and works across different screen sizes.

## Setup Instructions (Run Locally)

1. **Clone or download** this repository.
2. **Option A - Open directly:**  
   Open `index.html` in your browser (double-click or drag into the browser window).
3. **Option B - Local server (recommended for full behavior):**  
   From the project root, run a simple HTTP server:
   - **Python 3:** `python3 -m http.server 8000`  
   - **Node (npx):** `npx serve .`  
   Then visit `http://localhost:8000` (or the port shown in the terminal).
4. **Option C - Use a "Live Server" plugin/extension in VSCode:**
